# Raw Sources

Store source material here.

Rules:

- Treat files in this folder as immutable source material.
- Do not rewrite sources during wiki maintenance.
- If a source needs cleanup, create a cleaned copy and keep the original.
- Every source used by the wiki should have a corresponding note in [[wiki/sources/README|Source Notes]].
