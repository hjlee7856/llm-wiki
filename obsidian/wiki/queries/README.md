# Queries

Saved answers, analyses, comparisons, and explorations.

Use this folder when a user question produces reusable knowledge.

This folder is part of the compiled knowledge layer.
Do not treat saved query notes as chat transcripts; rewrite them as reusable wiki assets.

Typical examples:

- a comparison that will likely be asked again
- a synthesis across multiple source notes
- a decision memo with explicit evidence and caveats
- a project architecture summary produced from repeated Q&A
